# Mago en la Selva

Este proyecto es un minijuego desarrollado en Unity donde el jugador controla un mago que debe sobrevivir en un entorno de selva evitando enemigos.

## Mecánicas
- Movimiento del personaje
- Aparición de enemigos
- Sistema de vidas
- Puntuación y tiempo

## Controles
- Movimiento con teclado
- Evitar enemigos

## Tecnologías
- Unity
- C#
